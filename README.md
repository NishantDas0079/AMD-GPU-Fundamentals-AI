## 📖 Overview

This repository contains comprehensive study notes and a personal reference guide compiled from AMD's official webinars and technical documentation on **GPU Fundamentals & AI Workloads**.

It covers the complete journey — from AMD's hardware philosophy and ROCm software ecosystem to deep architectural details of CDNA GPUs (MI300 series) and practical AI compute patterns.

Perfect for students, researchers, HPC engineers, and AI practitioners looking to deeply understand AMD's GPU ecosystem as a strong alternative to NVIDIA CUDA.

---

## ✨ Key Topics Covered

- **AMD vs NVIDIA** — CUDA vs HIP vs ROCm
- **GPU Architecture** — RDNA vs CDNA, Compute Units, Wavefronts, MFMA
- **Memory Hierarchy** — Registers, LDS, L1/L2, Infinity Cache, HBM3
- **Performance Analysis** — Roofline Model, Latency Hiding, Occupancy
- **AI Workloads** — GEMM, Convolution, Mixed Precision, LLM Inference
- **Programming Model** — HIP, hipify, Composable Kernel, Triton
